# 简明说明

核心脚本

- `four_class_save_time_periods.py`
  - 功能：对四类刺激（IC2、IC4、LC2、LC4）分别筛选 RR 神经元，提取并保存每类的时间段信息（如 IC2/IC4/LC2/LC4 的关键帧集合、基线与空白时间段），输出为 `.npy` / `.mat` 文件以供后续分析使用。

- `graph_RR.py`
  - 功能：基于 RR 子集构建相关性网络并计算网络指标（度、聚类系数、模块化、小世界性等），支持按刺激时间段分割并对比不同时间段的网络特性。
  - 要点：脚本顶部有开关控制（如是否只使用兴奋性神经元、采样大小、阈值扫描范围），结果以日志/数据表或图形形式保存到实验目录或指定输出目录。

- `RR_hub_matching.py`
  - 功能：随机采样神经元（可设置样本数），计算皮尔逊相关矩阵并做阈值扫描，识别 hub 节点并统计与 RR 神经元的重合比例，最后可将阈值扫描结果保存为 Excel 报表。

- `svm_classification.py`
  - 功能：SVM 分类管线（支持四分类或二分类映射、按兴奋/抑制分组训练、可选 PCA、选择连续帧数做特征），包括训练、测试、混淆矩阵与结果保存。
  - 要点：常见的配置参数（`USE_PCA`, `N_FRAMES`, `BINARY_CLASSIFICATION`）可以在脚本顶部直接修改。
